# GLOSSARY

## Provider

The organization that creates LLM models. A "provider of models" ... often a Huggingfaces organization.

**Examples:**

 - moonshotai
 - microsoft
 - stability-ai


## Model

A specific trained AI artifact

**Examples:**

  - Phi-4-Reasoning-Q4_K_M 
  - SD-3.5-Large-Turbo

## Capability

What model CAN DO

**Examples:**

 - TextToText
 - TextToImage
 - TextEmbedding
 - ImageEmbedding
 - TextToSpeech
 - SpeechToText
 - Vision

## Registry Key: 

Hugginfaces `org/model-id` lookup ID

**Examples:**

- unsloth/Phi-4-reasoning-GGUF
